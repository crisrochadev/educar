# I-Educar Frontend

Frontend Angular + PrimeNG para gestão escolar com módulos Dashboard, Escola, Alimentação e Estoque.

## Como rodar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o app:
   ```bash
   npm start
   ```

## Estrutura

- `src/app/core`: serviços e interceptor JWT
- `src/app/shared`: modelos compartilhados
- `src/app/features`: módulos de negócio
- `src/app/layout`: sidebar, topbar e layout principal
